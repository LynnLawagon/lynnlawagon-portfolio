# Lynn Angela Lawagon Portfolio

This repository contains the personal portfolio website for Lynn Angela Lawagon, a Computer Science student with experience in web development, Java applications, and AI-assisted systems.

## Contents

- `index.html` — portfolio homepage
- `style.css` — site styling
- `script.js` — interactive behavior and theme switching
- `img/` — image assets used in the portfolio

## Features

- Responsive portfolio landing page
- Dark / light theme switcher
- Navigation sections for Home, About, Projects, Credentials, and Contact
- External links to GitHub and social profiles

## Preview

Open `index.html` in a web browser to view the portfolio locally.

## Notes

This project is a static website and does not require a web server for basic previewing. For development or sharing, you can also use a local server such as Live Server in VS Code.
